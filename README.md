# Smart Landing Creator

**Smart Landing Creator** é uma ferramenta web que permite criar, editar e exportar **landing pages personalizadas** de forma rápida, responsiva e acessível, diretamente no navegador.

## Funcionalidades

- **Edição em tempo real:** atualize título, subtítulo, texto, botão, cor do botão, imagem e cor de fundo.
- **Preview dinâmico:** alterações refletidas instantaneamente no preview.
- **Salvamento local:** cada landing é armazenada no navegador, permitindo editar ou deletar posteriormente.
- **Exportação de imagem:** gera uma imagem da landing pronta para portfólio ou compartilhamento.
- **Serviço personalizado:** se o modelo padrão não atender, há opção de contato via WhatsApp para landing 100% personalizada.
- **Acessibilidade:** navegação via teclado, leitura de labels, roles e feedbacks em tempo real.
- **SEO básico:** campos de title e description para buscadores.

## Tecnologias

- HTML5 semântico
- CSS3 moderno (gradientes, sombras e responsividade)
- JavaScript puro (DOM, eventos, localStorage)
- [html2canvas](https://html2canvas.hertzen.com/) para exportação de imagens
- Bootstrap 4.5 para layout responsivo

## Como usar

1. Clone ou baixe o projeto.
2. Abra `index.html` no navegador.
3. Preencha os campos do formulário e veja o preview atualizado em tempo real.
4. Salve a landing no armazenamento local ou exporte como imagem.
5. Cada landing salva mantém cores, texto e imagem individualmente.

## Deploy

O projeto é compatível com **GitHub Pages**:

1. Faça o push do repositório para o GitHub.
2. Vá em **Settings → Pages**.
3. Configure a branch `main` e a pasta `/` como fonte.
4. Acesse sua landing em `https://<username>.github.io/<repo>`.

## Conceitos aplicados

- Desenvolvimento frontend moderno com foco em **UX/UI**.
- **Acessibilidade (A11Y):** uso de roles, labels, `aria-live` e navegação por teclado.
- **Performance:** preview e exportação de forma otimizada sem frameworks pesados.
- **Manutenção e escalabilidade:** código modular e claro, facilitando futuras melhorias.
- **Design centrado no usuário:** cores, botões e imagens totalmente customizáveis.
- **SEO e marketing digital:** campos dedicados para otimização em buscadores.
 

**Pronto para uso ou personalização completa via contato pelo WhatsApp no banner da página.**
